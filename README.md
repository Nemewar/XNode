# Help:
Comandos para una tabla de multiplicar

```
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Es la base de la tabla a multiplicar                   [number]
  -l, --listar   Lista los elementos de tabla         [boolean] [default: false]
  -h, --hasta    Hasta que numero se va a multiplicar la base
```
